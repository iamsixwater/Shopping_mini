# Mini Shopping Game

A simple clothes data filtering page with Vanilla JS.

## Technical Stack

- HTML
- CSS
- Vanilla JS

## Functions

- Fetch data from JSON file
- List clotes data in list form
- Scroll items with overflowing data
- Filter a clothes list with 6 options with below:
  - t-shirt / pants / skirt
  - blue / yellow / pink
